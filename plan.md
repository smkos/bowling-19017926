# Bowling Kata TDD Plan

## Cycle 1 (완료)
목표: 모든 롤이 거터(0핀)인 게임의 점수는 0이다.
- `Game` 클래스에 `roll(pins)`와 `score()` 메서드를 만든다.
- 20번 `roll(0)`을 호출한 후 `score()`가 0을 반환하는지 검증한다.

## Cycle 2 (완료)
목표: 스트라이크/스페어 없이 모든 프레임이 같은 핀 수를 기록하는 게임의 점수는 각 롤의 합이다.
- 20번 `roll(1)`을 호출한 후 `score()`가 20을 반환하는지 검증한다.

## Cycle 3 (완료)
목표: 스페어(한 프레임에서 2롤 합 10)가 있으면 해당 프레임 점수는 10 + 다음 롤 핀 수이다.
- 1프레임에서 스페어(5, 5)를 기록하고, 다음 롤에서 3핀, 이후 나머지 17롤은 모두 0으로 채운다.
- 총점이 10(스페어) + 3(보너스) + 3(다음 프레임 실제 점수) = 16임을 검증한다.

## Cycle 4 (진행 예정)
목표: 스트라이크(첫 롤에서 10핀)가 있으면 해당 프레임 점수는 10 + 다음 두 롤 핀 수이고, 프레임은 롤 1번으로 끝난다.
- 1프레임에서 스트라이크(10)를 기록하고, 다음 두 롤에서 3, 4핀, 이후 나머지 16롤은 모두 0으로 채운다.
- 총점이 10(스트라이크) + 3 + 4(보너스) + 3 + 4(다음 프레임 실제 점수) = 24임을 검증한다.
